# richer-audio-melody-parsing_work_description

## Summary
- Reworked local audio analysis to stop using a fixed four-note melody derivation and instead derive a variable-length melody sequence (8-32 notes) from byte-windowed signal characteristics.
- Added deterministic helpers for tempo estimation, melody extraction diversity enforcement, and key estimation from pitch-class scoring.
- Updated local MIDI generation typing to accept variable-length melody tuples generated by the improved analyzer.
- Expanded entrypoint unit tests to cover full-song melody richness expectations and all added helper-branch behaviors.
