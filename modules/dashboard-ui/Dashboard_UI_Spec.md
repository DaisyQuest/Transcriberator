# Dashboard_UI_Spec.md

## Scope
Web dashboard application for projects, uploads, job execution visibility, and export management.

## Responsibilities
- Project listing and creation UX.
- Upload flow and job mode selection.
- Job timeline/log/status visualization.
- Artifact discovery and download UX.

## Quality Gates
- UI must remain responsive under frequent status updates.
- Failure/retry actions must be explicit and safe.
- Local-first developer startup path required.
